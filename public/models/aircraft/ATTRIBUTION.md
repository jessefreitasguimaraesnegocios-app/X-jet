# Modelo 3D do avião

- **Arquivo:** `cesium-air.glb`
- **Origem:** [Cesium](https://github.com/CesiumGS/cesium) — `Apps/SampleData/models/CesiumAir/Cesium_Air.glb`
- **Licença:** Apache License 2.0 (mesma do repositório Cesium)

O modelo é **ilustrativo** (aeronave tipo Cessna usada nos exemplos do Cesium), não corresponde necessariamente à matrícula ou ao tipo ICAO exibido no ADS-B. O app ajusta escala por categoria (narrow / wide / regional) para aproximar o tamanho.
